☁ Configurar GitHub como banco de dados

Os usuários serão salvos no arquivo nuvyon_banco.json no seu repositório privado do GitHub. Qualquer pessoa que abrir o dashboard vai ler do mesmo lugar. Configure apenas na sua máquina (admin).

Como obter o token:
1. Acesse github.com/settings/tokens/new
2. Dê um nome (ex: nuvyon-banco)
3. Marque apenas repo (acesso a repositórios privados)
4. Clique Generate token e copie o token gerado
